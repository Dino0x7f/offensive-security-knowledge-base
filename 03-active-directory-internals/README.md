# Active Directory

## Overview

This section explores the architecture, components, authentication mechanisms, and security model of Microsoft Active Directory from an offensive and defensive security perspective.

Rather than focusing only on administration, this section explains **how Active Directory works internally**, how trust is established between systems, how authentication occurs, and how attackers abuse legitimate features to compromise enterprise environments.

Understanding Active Directory is essential for:

- Enterprise Penetration Testing
- Red Team Operations
- Active Directory Security Assessments
- Windows Privilege Escalation
- Lateral Movement
- Identity Security
- Incident Response
- Threat Hunting

## Security Perspective

Active Directory is more than a directory service—it is the **identity and trust infrastructure** of Windows enterprise environments.

This section examines Active Directory from the perspective of:

- How identities are created
- How authentication works
- How authorization decisions are made
- How trust is established
- How credentials are protected
- How attackers abuse built-in features
- How defenders secure enterprise identity systems

Each topic builds toward understanding both the internal operation of Active Directory and the techniques used during modern enterprise attacks.
