# Foundations

## Overview

The **Foundations** section covers the fundamental concepts required before studying offensive security, penetration testing, and advanced attack techniques.

The goal is to build a strong understanding of how systems, networks, web applications, and cryptographic mechanisms work from a security perspective.

These topics focus on understanding technologies before learning how attackers exploit them.

---

## Contents

### Networking Basics
Learn how computer networks communicate and how attackers interact with network infrastructure.

Topics include:
- Network fundamentals
- Common protocols
- Network attacks
- Network security
- Network tools

---

### Cryptography Basics
Understand how modern systems protect data and where cryptographic implementations commonly fail.

Topics include:
- Encryption
- Hashing
- Digital signatures
- Certificates & PKI
- TLS/SSL
- Cryptographic attacks

---

### Operating Systems Basics
Study how Windows and Linux operate internally from a security perspective.

Topics include:
- Linux fundamentals
- Windows fundamentals
- File systems
- Users and permissions
- Processes and memory
- Services
- System hardening

---

### Web Security Basics
Learn the core concepts behind web applications and their most common attack surfaces.

Topics include:
- HTTP/HTTPS
- Sessions & Cookies
- Authentication & Authorization
- OWASP Top 10
- Web vulnerabilities
- APIs
- CORS & SOP
- Input validation

---

## Purpose

These foundations provide the technical knowledge needed to understand:

- How systems work
- Where trust boundaries exist
- How attackers identify weaknesses
- Why common vulnerabilities occur
- How offensive techniques exploit real-world systems

---

## Learning Goal

Before learning **how to attack**, understand **how the technology works**.

Strong fundamentals make advanced offensive security concepts easier to understand and apply.
