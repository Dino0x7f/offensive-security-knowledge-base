# About Me

### Introduction

I am an offensive security learner with a primary interest in penetration testing, Active Directory security, web application security, and Red Team methodologies.

My focus is not limited to using security tools. Instead, I aim to understand the technologies behind modern infrastructures and how attackers interact with them during real-world engagements.

- Post Exploitation
- Threat Modeling
- Security Reporting

---

### Learning Philosophy

I believe that effective penetration testing begins with understanding systems rather than memorizing tools.

Tools evolve continuously.

Technologies change.

Methodologies improve.

However, strong technical fundamentals remain valuable regardless of the tools being used.

For this reason, I prioritize learning:

- system architecture
- authentication mechanisms
- operating system internals
- network communication
- attack methodologies
- defensive concepts

before focusing on automation or exploitation frameworks.

---

### Documentation Philosophy

This repository serves as my personal technical knowledge base.

Every document is written with the intention of explaining concepts clearly, organizing information logically, and improving long-term understanding.

The objective is not to create short notes, but to build structured documentation that can be revisited and expanded over time.
