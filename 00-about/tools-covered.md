# Tools Covered

### Purpose

This document provides an overview of the security tools studied throughout this repository.

The objective is not to memorize tool usage, but to understand when, why, and how each tool supports different phases of a security assessment.

---

Tools commonly used for information gathering and target discovery.

- Nmap
- Masscan
- RustScan
- Amass
- Subfinder
- Assetfinder
- Whois
- Dig
- DNSRecon

---

### Tool Selection Philosophy

Tools are selected according to the assessment objective rather than personal preference.

The methodology followed throughout this repository prioritizes:

- Understanding the underlying technology.
- Selecting the appropriate tool for the task.
- Validating findings manually whenever possible.
- Documenting observations and methodology.

Tools support the assessment process, but they do not replace technical understanding.

The list of covered tools will continue to evolve as new technologies are studied and additional practical experience is gained.
