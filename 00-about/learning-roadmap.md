# Learning Roadmap

### Purpose

This roadmap defines the structured learning path followed throughout this repository.

The objective is not to study topics randomly, but to build technical knowledge progressively, where each stage provides the foundation for the next.

---

### Phase 1 — Technical Foundations

Understanding the technologies before attempting to secure or attack them.

Topics include:

- Computer Networking
- Linux Fundamentals
- Windows Fundamentals
- Operating Systems
- Cryptography Basics
- Web Technologies
- Programming Fundamentals

Outcome

Develop a solid understanding of how modern systems operate.

---

### Phase 2 — Security Fundamentals

Learning the core principles of information security.

Topics include:

- Security Principles
- Authentication
- Authorization
- Network Security
- Web Security
- Common Vulnerabilities
- Risk Assessment

Outcome

Understand how security mechanisms are designed and where they fail.

---

### Phase 3 — Penetration Testing

Applying offensive techniques to identify and validate security weaknesses.

Topics include:

- Reconnaissance
- Enumeration
- Vulnerability Validation
- Exploitation Concepts
- Post Exploitation
- Reporting

Outcome

Develop a structured penetration testing methodology.

---

### Phase 4 — Enterprise Security

Studying enterprise environments and identity-based infrastructures.

Topics include:

- Windows Internals
- Active Directory
- Kerberos
- NTLM
- LDAP
- Enterprise Networks

Outcome

Understand enterprise authentication and trust relationships.

---

### Phase 5 — Attack Chains

Studying how individual techniques combine into complete attack scenarios.

Topics include:

- Web Attack Chains
- Network Attack Chains
- Active Directory Attack Chains
- Cloud Attack Chains
- Enterprise Attack Paths

Outcome

Think in terms of attack progression rather than isolated vulnerabilities.

---

### Phase 6 — Red Team Operations

Learning offensive operations in realistic enterprise environments.

Topics include:

- Initial Access
- Command and Control
- Lateral Movement
- Privilege Escalation
- Defense Evasion
- Operational Security (OPSEC)

Outcome

Understand complete offensive engagements.

---

### Phase 7 — Reporting and Consulting

Communicating technical findings effectively.

Topics include:

- Executive Summaries
- Technical Reports
- Risk Scoring
- Remediation Guidance
- Client Communication

Outcome

Transform technical work into actionable security recommendations.

---

### Continuous Learning

Cybersecurity evolves continuously.

This roadmap is reviewed and expanded as new technologies, attack techniques, and defensive strategies emerge.

Learning is considered an ongoing process rather than a fixed destination.
